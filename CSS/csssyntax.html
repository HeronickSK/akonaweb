CSS Syntax a selektory


CSS Syntax

Skupina pravidiel CSS pozostáva zo selektoru(Selector) a bloku deklarácie(Declaration):

Selektor ukazuje na prvok HTML, ktorý chcete štýlovať.

Blok deklarácie obsahuje jedno alebo viac vyhlásení oddelených bodkočiarkami.

Každé vyhlásenie obsahuje názov vlastníctva CSS(Property) a hodnotu() oddelenú dvojbodkou.

Deklarácia CSS vždy končí bodkočiarkou a bloky deklarácie sa nachádzajú vždy v množinových zátvorkách {}.

V nasledujúcom príklade budú všetky prvky zarovnané v strede s červenou farbou textu:

p { color: red; text-align: center; }

CSS selektory

CSS Selektor sa používa na "vyhľadanie" (alebo výber) prvkov HTML na základe ich prvkového názvu, ID, triedy, atribútu atď.

Prvkový selektor

Prvkový selektor vyberá prvky na základe prvkového názvu.

Môžete vybrať všetky prvky < p> (paragrafu) na stránke ako napríklad takto (v tomto prípade všetky prvky < p> (paragrafu) budú zarovnané v strede s červenou farbou textu):

p { color: red; text-align: center; }

ID selektor

ID selektor používa id atribút HTML prvku na vybratie špecifického prvku(elementu).

ID prvku by malo byť jedinečné v rámci celej stránky, takže id selektor sa používa na výber jedného jedinečného prvku!

Na vybratie prvku s konkrétnym ID, napíšte znak hashtag (#) nasledujúc s id elementu.

Pravidlo štýlu nižšie uvedeného sa použije na prvok HTML s id = "para1":

#para1 { color: red; text-align: center; }

Poznámka: Identifikátor (ID) nemôže začínať číslom!

Zoskupujúce selektory

Ak máte prvky s rovnakými definíciami štýlov, ako napríklad:

h1 { color: red; text-align: center; }

h2 { color: red; text-align: center; }

p { color: red; text-align: center; }

Bude lepšie zoskupiť selektory, aby sa minimalizoval kód.

Ak chcete zoskupiť selektory, oddeľte každý selektor čiarkou.

V nižšie uvedenom príklade sme zoskupili selektory z vyššie uvedeného kódu:

h1, h2, p { color: red; text-align: center; }

CSS komentáre

Komentáre sa používajú na vysvetlenie kódu a môžu pomôcť pri neskoršom editovaní zdrojového kódu.

Prehliadače ignorujú komentáre.

Komentár CSS začína /* a končí s */. Komentáre môžu zahŕňať aj viacero riadkov:

h1 { color: red;

/* Toto je komentár s jedným riadkom */
