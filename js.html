JavaScript môže meniť obsah HTML

Začnem týmto príkladom, ktorý v sebe obsahuje metódu na nájdenie elementu s id="zmena" a zmení jeho obsah na "Ahoj Javascript!"

document.getElementById("zmena").innerHTML = "Ahoj Javascript!"

Javascript príjma obidve formy uvodzoviek či už dve ako v prvom príklade, alebo len jednu ako v tomto druhom príklade:

document.getElementById('zmena').innerHTML = "Ahoj Javascript!"'

Vyskúšaj si to :

Tento text sa zmení.
