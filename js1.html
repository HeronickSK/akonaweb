< script > Tag

V HTML sa JavaScript stále vkladá medzi < script > a < /script >

Napríklad

< script > document.getElementById("príklad").innerHTML = Môj prvý Javascript" < /script >

V starých príkladoch JavaScriptu sa môžete stretnúť aj s tvarom < script type="text/javascript" >

JavaScript Funkcie a Udalosti

Funkcia, je určitá časť kódu JavaScriptu, ktorá môže byť pri zavolaní vypnutá

Napríklad keď sa stane určitá udalosť, ako keď napríklad použivatel klikne myšou.

JavaScript v < body > a < head >

V HTML súbore môže byť ľubovolný počet scriptov

Scripty sa môžu nachádzať v < body >, v < head >, alebo aj v oboch

Externé JavaScripty

Scripty môžu byť tak isto uložené v externých súboroch.

Externé scripty sú dobré, keď chceme rovnaký kód využiť voviacerých súboroch.

JavaScript súbory majú koncovku .js

Na to, aby sme mohli použiť externý JavaScript súbor, musíme vložiť jeho meno do zdroja < script > tágu:

< script > src="myScript.js" < script >

Pozor! externé scripty nemôžu obsahovať < script > tag.
