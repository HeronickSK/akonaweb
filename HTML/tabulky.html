HTML Tabuľky

Definovanie HTML tabuľky

Tabuľka HTML je definovaná značkou table.Každý riadok je definovaný značkou tr. Tabuľka záhlavia je definovaná značkou th. Predvolene sú hlavičky tabuľky tučné a sústredené. Tabuľka údajov / bunka je definovaná značkou td.

Meno Priezvisko Vek
Jill Smith 50
Eve Jackson 94
John Doe 80

Okraje tabuľky

Ak neurčíte hranicu pre tabuľku, bude zobrazená bez okrajov.Okraj sa nastavuje pomocou CSS:

table, th, td { border: 1px solid black; }

Ak chcete ohraničiť okraje do jedného okraja, pridajte vlastnosť CSS border collapse:

príklad: table, th, td { border: 1px solid black; border-collapse: collapse;}

Cell padding špecifikuje priestor medzi obsahom buniek a jeho hranicami.Ak neurčíte padding bunky tabuľky sa zobrazia bez padding(polstrovania).

príklad: th, td { padding: 15px;}

Poloha hlavičiek

Predvolene sú hlavičky tabuľky tučné a centrované. Pre nastavenie polohy hlavičky v bunke sa používa text-aligh.

príklad: th { text-align: left; }

Vzdialenost buniek tabulky

Medzeri medzi bunkami sa určujú cez border-spacing

príklad: table { border-spacing: 5px; } NEPLATÍ ak tabuľka má okraje zlúčené

Viac buniek v jednom stĺpci

Viac buniek sa určuje cez colspan, ktorý sa nepíše do CSS ale priamo do tabuľky(poz.2).

Meno úspechy
Jožko vie počítať do 5 vie variť párky

Viac buniek v jednom riadku

Viac buniek sa určuje cez rowspan, ktorý sa nepíše do CSS ale priamo do tabuľky(poz.3).

Meno: Jožko
úspechy: vie počítať do 5
vie uvariť párky

Pridanie titulku tabuľke

Pre pridanie titulku sa používa caption(poz.4).

Nastavenie špeciálnych stýlov pre konkrétnu tabuľku

Na špecifikovnie vlasností tabuľky sa pridá tabuľke id a vlastnosti sa napíšu do CSS(poz.5).
