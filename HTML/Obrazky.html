Obrázky

Obrázky sú dôležitou súčasťou webstránky a všeobecne dokážu zlepšiť dizajn a celkový vzhľad webstránky.

Syntax obrázkov

V HTML sú obrázky definované značkou <img>.

Značka <img> je prázdna, obsahuje iba atribúty a nemá uzavieraciu značku.

Atribút src špecifikuje adresu URL (webovú adresu) obrázka:

<img src="url">

alt atribút

Atribút alt poskytuje alternatívny text pre obrázok, ak ho používateľ z nejakého dôvodu nemôže zobraziť (kvôli pomalému pripojeniu, chybe v atribúte src alebo ak používateľ používa čítačku obrazovky).

Element <picture>

HTML5 uviedol prvok <picture>, aby pri určovaní zdrojov obrázkov pridal väčšiu flexibilitu. Element <picture> obsahuje niekoľko prvkov <source>, z ktorých každý odkazuje na rôzne zdroje obrazu. Týmto spôsobom môže prehliadač vybrať obraz, ktorý najlepšie vyhovuje aktuálnemu zobrazeniu a / alebo zariadeniu. Každý prvok <source> má atribúty popisujúce, kedy je jeho obraz najvhodnejší. Prehliadač použije prvý prvok <source> s zodpovedajúcimi hodnotami atribútov a ignoruje všetky nasledujúce prvky <source>.

Príklad:

Zamieňajte veľkosti prehliadača, aby sa zobrazili rôzne verzie načítania obrázkov v rôznych veľkostiach výrezov. Prehliadač hľadá prvý zdrojový prvok, kde sa mediálny dopyt zhoduje s aktuálnou šírkou zobrazenia používateľa, a načíta obrázok špecifikovaný v atribúte srcset.

Prvok img sa vyžaduje ako posledná podradená značka bloku deklarácie obrázkov. Prvok img slúži na poskytovanie spätnej kompatibility pre prehliadače, ktoré nepodporujú obrazový prvok, alebo ak sa nezhoduje so žiadnou zo zdrojových značiek.

Toto nie je podporované v IE12 a starších verziách alebo Safari 9.0 a starších.

Čítače obrazovky HTML

Čítače obrazovky HTML Čítačka obrazovky je softvérový program, ktorý číta kód HTML, konvertuje text a umožňuje používateľovi "počúvať(listen)" obsah. Čítačky obrazovky sú užitočné pre ľudí, ktorí sú zrakovo postihnutí alebo učia sa.

ZHRNUTIE KAPITOLY OBRÁZKY

  • Použite prvok HTML <img> na definovanie obrázka
  • Použite atribút HTML src na definovanie adresy URL obrázka
  • Použite atribút HTML alt, ak chcete definovať alternatívny text pre obrázok, ak ho nemožno zobraziť
  • Pomocou atribútov šírky a výšky HTML môžete definovať veľkosť obrázka
  • Použite vlastnosti šírky a výšky CSS na definovanie veľkosti obrázka (alternatívne)
  • Použite vlastnosť CSS float, aby ste nechali obrázok plávať
  • Použite prvok HTML <map> na definovanie mapy obrázkov
  • Pomocou prvku HTML <area> môžete definovať oblasti, na ktoré môžete kliknúť, na mape obrázkov
  • Pomocou atribútu usmerňovacieho prvku HTML <img> použite ukazovateľ obrázku
  • Pomocou prvku HTML <picture> zobrazte rôzne obrázky pre rôzne zariadenia
