CSS Intro


Čo je CSS?

Všeobecne povedané, CSS je nejaký zápis, ktorý určuje vzhľad (farby, dekoračné obrázky, rozmiestnenie prvkov) HTML dokumentu. CSS je skratka slov Cascading Style Sheets. Do slovenčiny by sme to preložili ako Tabuľky kaskádových štýlov. V praxi sa potom častejšie používajú tieto názvy: Kaskádové štýly, štýly, CSS súbor (y).

Súbor s kaskádovými štýlmi má príponu css. Sám o sebe je k ničomu. Jeho funkčnosť sa prejaví až po prepojení s HTML súborom. A prečo vlastne kaskádové? Pretože pri písaní CSS súboru sa využívajú tzv. kaskády.

HTML dokumentu môžeme rôznymi CSS súbormi zabezpečiť rôzne vhľady a naopak rôznym HTML súborom môžeme pomocou rôznych CSS súborov zaistiť rovnaký vzhľad.

V tejto súvislosti by som však chcel upozorniť na to, že by sme sa mali držať nejakej predpísanej štruktúry HTML dokumentu. Aj keď pomocou CSS môžeme vzhľad HTML dokumentu ľubovoľne meniť, mali by sme mať na pamäti, že v prehliadači môžeme štýly vypnúť. Ak by sme potom mali napríklad hlavný nadpis h1 až na konci zdrojového kódu, orientácia v takomto dokumente by asi nebola najrúžovejšia.

Prečo používať CSS?

CSS sa používa na definovanie štýlov pre vaše webové stránky, vrátane návrhu, rozloženia a variácií zobrazenia pre rôzne zariadenia a veľkosti obrazovky.

CSS vyriešilo veľký problém

Jazyk HTML nebol NIKDY určený na to, aby obsahoval značky na formátovanie webovej stránky!

HTML bol vytvorený na opis obsahu webovej stránky, ako napríklad:

< h1> Toto je nadpis < /h1>

< p> Toto je odsek. < /p>

Keď boli do špecifikácií HTML 3.2 boli pridané tagy ako < font> a farebné atribúty, pre webových vývojárov sa začala nočná mora. Vývoj rozsiahlych webových stránok, kde bolo na každej stránke pridané písmo a farebné informácie, sa stal dlhým a nákladným procesom.

Na vyriešenie tohto problému vytvorilo konzorcium World Wide Web (W3C) CSS.

CSS odstránilo formátovanie štýlov z HTML stránky!

CSS šetrí veľa práce!

Definície štýlu sa zvyčajne ukladajú do externých súborov .css.

S externým súborom šablón so štýlmi môžete zmeniť vzhľad celej webovej lokality zmenou len jediného súboru! 

body
{
  background-color: lightblue;
}
h1
{
  color: white;
  text-align: center;
}   
p  
{
  font-family: verdana;
  font-size: 20px;
}
